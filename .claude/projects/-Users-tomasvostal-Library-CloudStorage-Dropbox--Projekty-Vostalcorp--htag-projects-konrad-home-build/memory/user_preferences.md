- Nepoužívat znak — (em dash) v textech na webu, nahradit čárkou/tečkou
- Web je čistě o dřevostavbách (ne střechy, vazníky)
- Cíl webu: nezávazná poptávka/konzultace zdarma, uklidnit návštěvníky
- CTA text: "Nezávazná poptávka zdarma" + "K ničemu se nezavazujete"
- Menu "Dřevostavby na klíč" (ne jen "Dřevostavby")
- Cílová skupina: rodiny 30-50 let s malými dětmi
- Admin: položky vybavení po jedné (ne textarea), WYSIWYG editor pro popisy
- Soft delete (koš) místo trvalého mazání
- Koš a log aktivit schovat do Nastavení (ne v hlavní navigaci)
- Loga partnerů: čtvercová (Solodoor, Ptáček) větší než široká (PRO-DOMA, Supellex)
- Commit messages v češtině

**Why:** Klient (Lubomír Konrad) bude web spravovat sám, musí to být intuitivní.
**How to apply:** Vždy preferovat jednoduchost UI, čistý design, české texty.
